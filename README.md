# Selenium Login Automation Project

This project demonstrates basic automation testing using Selenium with Python.

## Test Cases Covered:

* Valid Login Test
* Invalid Login Test

## Tools Used:

* Python
* Selenium WebDriver
* WebDriver Manager

## Website Tested:

https://the-internet.herokuapp.com/login

## Features:

* Automated login functionality
* Validation of success and error messages
* Use of explicit waits for dynamic elements
